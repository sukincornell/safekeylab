Privacy by Design Principles

üî¨ Data Minimization

Aegis processes only the minimum data necessary for PII detection. Text is analyzed in-memory and immediately purged - no storage, no logs, no persistence.

üéØ Purpose Limitation

Data is used exclusively for PII detection and protection. No secondary use, no analytics, no model training on customer data.

Zero data retention policy. All processing happens in ephemeral compute instances that are destroyed after each request.

Maximum security settings enabled by default. All API calls require authentication, encryption, and are subject to rate limiting.
