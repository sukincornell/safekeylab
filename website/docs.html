Quick Start (60 Seconds)

The fastest way to protect your AI application from leaking PII:

Step 1: Get SDK Access

# SDK available through private registry
# Contact sales@aegis.ai for access

# After approval, you'll receive:
# - Private PyPI repository URL
# - npm registry credentials
# - API keys for authentication

Step 2: Get Your API Key

Contact sales at aegis.ai to get your API key. Test environment available for POC.

Step 3: Add One Line of Code

# BEFORE - Dangerous (sends PII to OpenAI)
import openai

user_input = "My SSN is 123-45-6789"
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": user_input}]
)

# AFTER - Protected (PII removed before OpenAI)
import openai
from aegis import protect

user_input = "My SSN is 123-45-6789"
safe_input = protect(user_input)  # <-- One line added
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": safe_input}]
)
✅ That's it! Your application is now HIPAA, GDPR, and PCI compliant. SSNs, credit cards, and all PII are automatically removed before reaching OpenAI.

Include your API key in all requests:

For enterprise customers with enhanced security requirements:

client = AegisClient(
    cert_path="/path/to/client.crt",
    key_path="/path/to/client.key",
    auth_method="mtls"

Your First Request

Basic Protection

from aegis import protect

# Input with multiple PII types
text = """
Patient: John Smith
SSN: 123-45-6789
Diagnosis: Depression
Credit Card: 4532-1234-5678-9012
"""

# Protect the text
protected = protect(text)

print(protected)
# Output:
# Patient: [NAME]
# SSN: [SSN_REDACTED]
# Diagnosis: [MEDICAL_CONDITION]
# Credit Card: [CREDIT_CARD_REDACTED]

With Options

protected = protect(text, {
    "mode": "strict",           # strict | balanced | minimal
    "method": "redaction",      # redaction | masking | tokenization
    "preserve_format": True,     # Keep text structure
    "entity_types": ["ssn", "credit_card"],  # Specific types only
    "custom_patterns": [{
        "name": "employee_id",
        "pattern": r"EMP\d{6}"
    }]
})

How It Works

Aegis uses a multi-layer approach to achieve 100% PII detection accuracy:

1. Pattern Recognition

Regex patterns for structured data like SSNs, credit cards, phone numbers

2. Machine Learning

Transformer models (DeBERTa) for contextual understanding of names, addresses

3. Dictionary Matching

Medical terms, financial terminology, legal phrases

4. Context Analysis

Understanding surrounding text to identify sensitive information

⚠️ Important: Aegis processes data in-memory only. We never store, log, or train on your data.

OpenAI Integration

ChatGPT / GPT-4

import openai
from aegis import protect_openai

# Automatic protection for all OpenAI calls
openai = protect_openai(openai)

# Now all calls are automatically protected
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "My SSN is 123-45-6789"}]
)
# OpenAI receives: "My SSN is [REDACTED]"

Embeddings

# Protect embeddings to prevent PII in vector databases
embedding = openai.Embedding.create(
    input=protect("Customer John Smith, card 4532-1234-5678-9012"),
    model="text-embedding-ada-002"
)

Claude/Anthropic Integration

from anthropic import Anthropic
from aegis import protect_anthropic

# Wrap Anthropic client
client = protect_anthropic(Anthropic())

# Protected automatically
message = client.messages.create(
    model="claude-3-opus",
    messages=[{
        "role": "user",
        "content": "Process payment for SSN 123-45-6789"
    }]
)

Streaming Support

Real-time Protection

# Protect streaming responses
import openai
from aegis import protect_stream

stream = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[{"role": "user", "content": user_input}],
    stream=True
)

# Protect each chunk in real-time
for chunk in protect_stream(stream):
    print(chunk.choices[0].delta.content, end="")

API Endpoints

POST /v3/protect

Main endpoint for PII protection

{
  "text": "string",           // Text to protect
  "mode": "strict",          // strict | balanced | minimal
  "method": "redaction",     // redaction | masking | tokenization
  "preserve_format": true,   // Maintain text structure
  "entity_types": ["ssn"],   // Optional: specific types
  "webhook_url": "string"    // Optional: async callback
}

POST /v2/detect

Detection only (no modification)

{
  "text": "string",
  "entity_types": ["all"]    // Types to detect
}

// Response
{
  "entities": [
    {
      "type": "ssn",
      "value": "123-45-6789",
      "start": 10,
      "end": 21,
      "confidence": 0.99
    }
  ]
}

Error Handling

from aegis import AegisClient, AegisError

try:
    protected = client.protect(text)
except AegisError as e:
    if e.code == "RATE_LIMIT":
        # Wait and retry
        time.sleep(1)
        protected = client.protect(text)
    elif e.code == "INVALID_API_KEY":
        # Check your API key
        print("Invalid API key")
    else:
        # Log and handle
        logger.error(f"Aegis error: {e}")
